1. Задача курсовой работы:
  В курсовой работе была поставлена задача выполнить верстку одностроничного веб-сайта по предоставленному макету и загрузить ее на свой репозиторий в GitHub.

2. Ресурсы для реализации поставленной задачи:

 По макету из Figma (https://www.figma.com/file/8TqeYoJVzCqebB3abpzoN7/Free--Landing--Page-Template?node-id=0%3A1) 
  в редакторе кода Visual Studio Code 
    с помощью HTML5, CSS, JavaScript 
      был сверстан одностраничный сайт 
        и загружен на мой репозиторий в GitHub (https://github.com/Ann-Fox).

  HTML (HyperText Markup Language — «язык гипертекстовой разметки») — самый базовый строительный блок Веба.
 
  CSS (Cascading Style Sheets) —  язык иерархических правил (таблиц стилей), используемый для представления внешнего вида документа, написанного на HTML. CSS описывает, каким образом элемент должен отображаться на экране, на бумаге, голосом или с использованием других медиа средств.

  JavaScript® (часто просто JS) — это легковесный, интерпретируемый или JIT-компилируемый, объектно-ориентированный язык. Наиболее широкое применение находит как язык сценариев веб-страниц, но также используется и в других программных продуктах, например, node.js. 
  
  Visual Studio Code — это легкий, но мощный редактор исходного кода, который работает на рабочем столе и доступен для Windows, macOS и Linux. Он поставляется со встроенной поддержкой JavaScript, TypeScript и Node.js и имеет богатую экосистему расширений для других языков (таких как C++, C#, Java, Python, PHP, Go) и сред выполнения (таких как .NET и Unity).

  GitHub — это платформа для размещения кода для контроля версий и совместной работы, это служба хостинга репозитория Git. Хотя Git — это инструмент командной строки, GitHub предоставляет графический веб-интерфейс. Он также обеспечивает контроль доступа и функций совместной работы, основные инструменты управления задачами для каждого проекта.

  Figma - кроссплатформенный графический редактор для совместной работы. В ней можно создавать: макеты сайтов, приложений и кликабельные прототипы с анимацией отдельные элементы интерфейса: иконки, кнопки, формы и многое другое.

  Локальный сервер - для тестирования и отладки сайта на своем компьютере.
